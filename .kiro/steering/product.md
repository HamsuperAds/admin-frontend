# Product Overview

This is an admin frontend application for HamSuper, a classified ads platform. The admin panel allows administrators to manage users, advertisements, and platform operations.

## Key Features

- Admin authentication and verification
- Dashboard for managing classified ads
- User management and moderation
- Ad categorization and status management
- Role-based access control (user, admin, moderator)

## Target Users

Platform administrators and moderators who need to oversee and manage the HamSuper classified ads marketplace.
